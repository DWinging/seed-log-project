## 📑 Seed-Log API 명세서 v1.0

### 1. 게시글 (Post) 관련 API
| 기능 | 메서드 | 엔드포인트 | 설명 |
| :--- | :--- | :--- | :--- |
| **게시글 작성** | `POST` | `/api/posts` | 새로운 포스트 생성 (카테고리 ID, 태그 목록 포함) |
| **게시글 목록 조회** | `GET` | `/api/posts` | 전체 게시글 리스트 조회 (Pageable 기반 페이징) |
| **게시글 상세 조회** | `GET` | `/api/posts/{id}` | 특정 ID의 게시글 상세 내용 및 연결된 태그 조회 |
| **게시글 수정** | `PUT` | `/api/posts/{id}` | 특정 ID의 게시글 수정 (제목, 내용, 카테고리, 태그) |
| **게시글 삭제** | `DELETE` | `/api/posts/{id}` | 특정 ID의 게시글 삭제 (매핑 데이터 자동 삭제) |

### 2. 카테고리 (Category) 관련 API
| 기능 | 메서드 | 엔드포인트 | 설명 |
| :--- | :--- | :--- | :--- |
| **카테고리 목록 조회** | `GET` | `/api/categories` | 저장된 모든 카테고리 목록 조회 |
| **카테고리 생성** | `POST` | `/api/categories` | 새로운 관리용 카테고리 추가 |
| **카테고리 변경** | `PUT` | `/api/categories/{id}` | 특정 ID의 카테고리 이름 변경 |
| **카테고리 삭제** | `DELETE` | `/api/categories/{id}` | 특정 ID의 카테고리 삭제 및 매핑 해제 |

### 3. 태그 (Tag) 관련 API
| 기능 | 메서드 | 엔드포인트 | 설명 |
| :--- | :--- | :--- | :--- |
| **태그 목록 조회** | `GET` | `/api/tags` | 시스템에 등록된 모든 태그 목록 조회 |
| **태그별 게시글 조회** | `GET` | `/api/tags/{id}/posts` | 특정 태그가 달린 게시글 리스트 조회 |
